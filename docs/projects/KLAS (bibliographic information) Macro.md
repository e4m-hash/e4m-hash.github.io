## 프로젝트 개요

KLAS(Korea Library Automation System)는 공립도서관에서 사용되는 서지정보 관리
웹 페이지로 해당 repository 는 반복적인 서지 정보 등록 작업을 자동화하기 위해 개발된 매크로 기반 프로젝트입니다. KLAS에서 발생하는 단순 반복 작업을 자동화함으로써 사용자의 편의성을 높이고 시간을 절약하는 것을 목표로 합니다.

## 주요 특징

- **서지 정보 자동 등록**: 수작업으로 처리하던 서지 정보 입력을 자동화하여 업무 효율 향상
- **GUI 제공**: PyQt5 기반의 직관적인 인터페이스로 비개발자도 손쉽게 매크로 실행 가능
- **브라우저 자동화**: Selenium과 Chrome WebDriver를 활용해 웹 기반 KLAS 시스템을 직접 조작
- **HTML 요소 제어**: 웹 페이지의 DOM 요소를 탐색·클릭·입력하여 실제 사용자 동작을 그대로 재현
