# 상대 위험도 (Relative Risk, RR)

|           | 결과 (유)   | 결과 (무)   | 합        |
| --------- | -------- | -------- | -------- |
| 실험군 (노출)  | $n_{11}$ | $n_{12}$ | $n_{1+}$ |
| 대조군 (비노출) | $n_{21}$ | $n_{22}$ | $n_{2+}$ |
| 합         | $n_{+1}$ | $n_{+2}$ | $n$      |
- 정의
$$
RR\;=\;\frac{\text{실험군의 위험률}}{\text{대조군의 위험률 }}\;=\;\frac{n_{11}/n_{1+}}{n_{21}/n_{2+}}
$$
- 해석
	- RR = 1 : 위험 차이 없음
	- RR > 1 : 실험군이 더 위험
	- RR < 1 : 실험군에서 위험 감소

- 적용 연구 : 코호트 연구, 임상시험
	- 대조군의 수가 연구자에 의해 결정되므로 사례 대조 연구 적용은 부적절하다.
###### 사례 대조 연구 - 상대 위험도
- 사례 대조 연구 특성
	- 질병 상태( 사례 / 대조 ) 를 먼저 정하고, 이후 노출 여부 조사
		- 모집단의 질병 발생 확률 알 수 없음
	- 사례/대조 군의 비율은 연구자가 임의로 정한 샘플링 비율
		- 예 : 실제 질병 유병률 1%, 사례 50명/대조 50명 -> 발생률 왜곡

- 대안 : 오즈 비
	- 샘플링 비율에 상관 없이 노출의 오즈비는 모집단과 동일

# 오즈비 ( Odds Ratio, OR )
|           | 결과 (유)   | 결과 (무)   | 합        |
| --------- | -------- | -------- | -------- |
| 실험군 (노출)  | $n_{11}$ | $n_{12}$ | $n_{1+}$ |
| 대조군 (비노출) | $n_{21}$ | $n_{22}$ | $n_{2+}$ |
| 합         | $n_{+1}$ | $n_{+2}$ | $n$      |

- Odds = $\frac{\text{사건 발생 확률}}{\text{사건 미발생 확률}}$
- Odds Ratio

$$
\begin{align}
OR\;&=\;\frac{\text{실험군에서의 오즈}}{\text{대조군에서의 오즈}} \\\\
\;&=\;\frac{(n_{11}/n_{+1})/(n_{21}/n_{+1})}{(n_{12}/n_{+2})/(n_{22}/n_{+2})} \\\\
\;&=\;\frac{n_{11}n_{22}}{n_{12}n_{21}}
\end{align}
$$

- 해석
	- OR = 1 : 연관성 없음
	- OR > 1 : 위험 요인과 사건 양의 연관
	- OR < 1 : 위험 요인과 사건 음의 연관
- 적용 연구 : 사례-대조 연구, 코호트 연구 모두 사용 가능

# RR/OR 의 신뢰구간

## CI(RR)

$$
\begin{align}
CI(RR)\;=\;[RR\;*\;e^{-z_{a/2\sqrt{\hat{Var}(\ln RR)}}},RR\;*\;e^{z_{a/2\sqrt{\hat{Var}(\ln RR)}}}]
\end{align}
$$
단, $\hat{Var}(\ln RR)\;=\;\frac{1}{n_{11}}\;-\;\frac{1}{n_{1+}}\;+\;\frac{1}{n_{21}}\;-\;\frac{1}{n_{2+}}$

## CI(OR)
$$
\begin{align}
CI(OR)\;=\;[OR\;*\;e^{-z_{a/2\sqrt{\hat{Var}(\ln OR)}}},OR\;*\;e^{z_{a/2\sqrt{\hat{Var}(\ln OR)}}}]
\end{align}
$$

단, $\hat{Var}(\ln OR)\;=\;\frac{1}{n_{11}}\;+\;\frac{1}{n_{12}}\;+\;\frac{1}{n_{21}}\;+\;\frac{1}{n_{22}}$

### 로그 변환과 분산 추정
#### 로그 변환
- RR/OR 은 비율이라 음수가 될 수 없다
- log(RR)/log(OR) 은 정규분포 근사에 유리
- 따라서, CI 는 지수함수로 되돌려서 계산

# 연관성의 검증
- 카이제곱 검정
	-> 샘플수 불충분 : 피셔의 정확 검정법

## 카이제곱 검정
### 귀무가설
$H_{01}$ : 아스피린 복용시와 폴라시보 복용시의 심근경색의 발병 비율은 같다.
-> 동질성 검정 (homogeneity test)
$H_{02}$ : 아스피린 복용 여부와 심근경색 발볍 여부는 서로 독립이다.
-> 독립성 검정 (independence test)

### 동질성 검정
| 그룹/결과    | 유        | 무        | 합        |
| -------- | -------- | -------- | -------- |
| 실험군(노출)  | $n_{11}$ | $n_{12}$ | $n_{1+}$ |
| 대조군(비노출) | $n_{21}$ | $n_{22}$ | $n_{2+}$ |
| 합        | $n_{+1}$ | $n_{+2}$ | $n$      |
- $I=2$ , $J=2$ 일때, 2x2 분할표
- $p_{ij}$ : i 번째 그룹에서 j 번째 결과가 될 확률
- $H_o\;:\;p_{1j}\;=\cdots=\;p_{ij}\;=\cdots=\;p_{ij}$
- $\hat p_j = {n{+j} \over n}$

귀무 가설이 맞다면 모든 처리 그룹에 대해 j 번째 결과의 확률이 모두 $p_j$ 와 같다.
고로 i 번째 그룹의 j번째 결과에 대한 기대도수는
- $E_{ij} = n_i + \hat p_j = {n_i + n_{+j} \over n }$

관측도수와 기대도수의 차이가 클 수록 귀무가설을 기각하는 강한 증거가 된다.
$$
\begin{align}
\chi^2\;=\;&\sum_{i=1}^I\sum_{j=1}^J\frac{(n_{ij}-E_{ij})^2}{E_{ij}} \\
 = & \sum_{i=1}^I\sum_{j=1}^J{(n_{ij} - n_i + n_{+j} /n)^2 \over  (n_{i+} n_{+j}) /n}
\end{align}
$$
이로 카이제곱 통계량은 위와같이 정의된다.
귀무가설 하에서 이 통계량은 자유도 (I-1)(J-1)인
카이제곱 분포를 따른다는 것을 이용하여 검정한다.
### 독립성 검정
동질성 검정과 유도과정이 약간 다르지만 결과적으로 동일하다.
## 피셔

-> 각 셀(30%)의 관측 도수가 5 미만인 경우
-> 표본 크기가 작은 경우

초기하 분포 :
	- 행합, 열합이 고정된 2x2 표의 확률 분포

p : 연가설 하에서 관측된 통계량보다 극단적인 값이 나올 확률

이 검정법에서는 셀 하나가 변해도 모든 셀의 값이 변하므로
한 셀이 변하는 확률만 구하면 된다.

### 초기하 분포 (HyperGeoMatrix Distribution)
유한 모집단에서 복원하지 않고 추출할때, 성공 갯수의 분포
즉, 특정 셀 값은 초기하 분포를 따름

|           | 결과 (유)   | 결과 (무)   | 합        |
| --------- | -------- | -------- | -------- |
| 실험군 (노출)  | $n_{11}$ | $n_{12}$ | $n_{1+}$ |
| 대조군 (비노출) | $n_{21}$ | $n_{22}$ | $n_{2+}$ |
| 합         | $n_{+1}$ | $n_{+2}$ | $n$      |


$$
\begin{align}
P(n_{11}=k) =\\\\
\frac{\begin{bmatrix} n_{+1} \\ k \end{bmatrix}\begin{bmatrix} n_{+2} \\ n_{1+}-k \end{bmatrix}}{\begin{bmatrix} n \\ n_{1+} \end{bmatrix}}
\end{align}
$$

|           | 결과 (유) | 결과 (무) | 합   |
| --------- | ------ | ------ | --- |
| 실험군 (노출)  | 1(a)   | 4      | 5   |
| 대조군 (비노출) | 3      | 2      | 5   |
| 합         | 4      | 6      | 10  |
-> a 가 1일때의 테이블
-> 초기하분포를 따른다.

a : 표의 (합) 중 최소값까지의 값만 가질 수 있다.
-> 2의 값을 가지지 못하는 이유
-> 관측된 테이블보다 작거나 같은 확률을 가진 테이블을 구해야한다.
-> 즉, 1일때의 $P(A=a)$ 보다 작거나 같은 확률을 가져야만한다.

## 맥니마 검정
|        | 처리후(Y)   | 처리후(N)   | 합        |
| ------ | -------- | -------- | -------- |
| 처리전(Y) | $n_{11}$ | $n_{12}$ | $n_{1+}$ |
| 처리전(N) | $n_{21}$ | $n_{22}$ | $n_{2+}$ |
| 합      | $n_{+1}$ | $n_{+2}$ | $n$      |
- ex) 소화제 먹기 전/후 - 소화불량 Y / N
- 두 표본은 독립이 아니다. 

가설 :
- 조건 1에서의 반응률 $p_1$ , 조건 2에서의 반응률 $p_2$
- $H_0\;:\;p_1=p_2\quad,\quad H_0\;:\;p_1\neq p_2$
- $\hat p_1 = {n_{1+} \over n}$ , $\hat p_2 = {n_{+1} \over n}$ 

검정통계량 :
$$
\begin{align}
Q_M={(n_{12}-n_{21})^2 \over (n_{12}+n_{21}} \sim \chi^2(1)
\end{align}
$$
연속성 보정된 검정통계량 :
$$
\begin{align}
Q_{M(2)}={\lvert (n_{12} - n_{21} - 1)^2 \rvert \over (n_{12} + n_{21})}
\end{align}
$$

### 개념
연속형 변수의 분석에서 짝지은 t-검정이 있었던 것처럼
도수를 구한 데이터에서도 짝지은 표본에 대해서 분석방법을 달리 고려해야한다.

한 샘플에 대해 처리 전,후를 비교한 문제에 사용 할 수 있다.

두 조건에서 일치한 쌍의 개수는 고려하지 않고 일치하지 않은 쌍의 개수에 근거하여
검정 통계량을 구한다.

처리 전에 양성 반응이었다가 처리 후에 음성이 된 사람이
처리 전에 음성이었다가 처리 후에 양성이 된 사람보다 현저히 많다면
처리 전 후의 반응률은 차이가 있는 것이고
귀무가설은 기각되어야한다.

## 코크란 맨텔 헨젤 검정 (CMH)
| K층    | 반응(Y)     | 반응(N)     | 합         |
| ----- | --------- | --------- | --------- |
| 처리(1) | $n_{k11}$ | $n_{k12}$ | $n_{k1+}$ |
| 처리(2) | $n_{k21}$ | $n_{k22}$ | $n_{k2+}$ |
| 합     | $n_{k+1}$ | $n_{k+2}$ | $n_k$     |
- 층화자료(Stratified Data) 에서 교란 변수 효과 보정
- 구조 : 2x2 by k 분할표
- $H_0$ : 층을 통제한 후에도 두 변수간 연관성 없음
	$H_0$ : $p_1=p_2$ , $H_1$ : $p_1 \neq p_2$

$$
\begin{align}
Q_{CMH}\;=\;\frac{(\sum_{k=1}^Kn_{k11} - \sum_{k=1}^K(n_{k11}|H_0))}{\sum_{k=1}^K Var(n_{k11}|H_0)}\sim \chi^2(1)
\end{align}
$$

### 개념
여러 개의 그룹이 있을때, 두 처리간 반응률의 차이가 있는지 검정한다.
독립된 K개의 그룹이 있을때, 그룹의 효과를 제어한 반응률의 차이가 있는지 검정하는 방법
$k=(1,2,\cdots,K)$ 번째 층의 처리와 반응 간의 관계
K 개의 층이 있고 처리 1 에서의 반응률을 $p_1$ , 처리 2에서의 반응률을 $p_2$ 라고 할때
처리 간 전체적인 반응률의 차이가 있는지를 검정하기 위한 귀무가설은 다음과 같다.
$H_0$ : $p_1=p_2$ , $H_1$ : $p_1 \neq p_2$

$E(n_{k11}|H_0)={n_{k1+}n{k+1} \over n_k}$ 이고 $Var(n_{k11}|H_0)={n_{k1+}n_{k2+}n_{k+1}n_{k+2} \over n_{k}^2(n_k -1)}$ 이다.

통계량 $Q_{CMH}$ 이 귀무가설 하에서 근사적으로 자유도 1 을 갖는 카이제곱 분포를 따른다는 것을
이용하여 가설 검정한다.

# 진단법의 평가

| 진단/질병 | 양성  | 음성  |
| ----- | --- | --- |
| 양성    | TP  | FP  |
| 음성    | FN  | TN  |

- 민감도 - 특이도 : 반비례 관계
- 백분률 표시, 신뢰구간 계산 가능
- 100%(1.0) 이 가장 좋은 값
- 쉬운 치료 질병 -> 민감도 중요
- 하드 치료 질병 -> 특이도 높은 검사
## 민감도 
- 양성 찾기 
- TPR(True Positive Rate)
$$
\begin{align}
\frac{TP}{TP\;+\;TN}
\end{align}
$$
## 특이도
- 음성 찾기 
- FPR(False Positive Rate) = 1 - 특이도
$$
\begin{align}
\frac{TN}{FP\;+\;TN}
\end{align}
$$

## 정확도
$$
\begin{align}
\frac{TP\;+\;TN}{N}
\end{align}
$$
-> 전체 테스트 군에서 올바르게 진단된 경우

## 유병률
$$
\begin{align}
\frac{TP\;+\;FN}{N}
\end{align}
$$
-> 전체 테스트 군에서 질병을 가지고 있는 경우
