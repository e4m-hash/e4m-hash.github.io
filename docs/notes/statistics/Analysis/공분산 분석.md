#### 공분산 분석표 (제 3종 제곱합)
```R
Anova(lm(y ~ x + method, data=data), Type="3")
```

|             |  Sum Sq   | Df  |  F value  | Pr(>F)  |
| :---------: | :-------: | :-: | :-------: | :-----: |
| (Intercept) | 1978.6758 |  1  | 2455.9081 | < 0.001 |
|      x      |  47.3049  |  1  |  58.7143  | < 0.001 |
|   method    |  83.1307  |  2  |  51.5904  | < 0.001 |
|  Residuals  |  4.0284   |  5  |    NA     |   NA    |

제 3종 제곱합은 $SS(x\;|\;method) + SS(method\;|\;x)$ 가 된다.
F 값 51.59, p 값이 유의수준 0.05 에서 매우 작으므로,
따라서 Method 의 효과는 초기 수치를 보정했을때, 유의하다는 것을 의미한다.

초기 시험성적에 대한 p 값은 매우 작으므로 유의수준 0.05 에서
초기 시험성적에 대한 효과가 0이라는 위 연구문제의 $H_{01}$ 귀무가설을 기각한다.

#### 모수 추정치 결과
```R
summary(lm(y ~ x + method, data=data))
```

|           | Estimate | Std. Error |   t    |    p    |
| :-------: | :------: | :--------: | :----: | :-----: |
| Intercept | 33.1037  |   0.6680   | 49.557 | < 0.001 |
|     x     |  0.1900  |   0.0248   | 7.663  | 0.0006  |
| Method 2  | -1.5866  |   0.7336   | -2.163 | 0.0829  |
| Method 3  |  5.5233  |   0.7333   | 7.532  | 0.0007  |

$$
\begin{align}
&\hat y_{1j}\;=\;\hat \beta_0+\hat \alpha_1+\hat \beta x_{1j}\;=\;33.1037+0+0.1900*x_{1j} \\
&\hat y_{2j}\;=\;\hat \beta_0+\hat \alpha_2+\hat \beta x_{2j}\;=\;33.1037-1.5866+0.1900*x_{2j} \\
&\hat y_{3j}\;=\;\hat \beta_0+\hat \alpha_3+\hat \beta x_{3j}\;=\;33.1037+5.5233+0.1900*x_{3j}
\end{align}
$$

교육 방법 1을 기준으로 유의수준 0.05에서 교육방법 2에 차이에 대한 p 값이 0.0829 로 유의하지 않음으로 차이가 없다.
또한 교육 방법 1을 기준으로 유의수준 0.05에서 교육방법 3에 차이에 대한 p 값이 0.0007 로 유의함으로 차이가 있다.
$$
\begin{align}
\text{not}\;H_{02} : \alpha_1\neq\alpha_2\quad\text{or}\quad\alpha_2\neq\alpha_3 \quad\text{or}\quad\alpha_1\neq\alpha_3
\end{align}
$$
