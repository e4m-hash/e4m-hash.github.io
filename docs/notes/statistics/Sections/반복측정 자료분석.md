# Repeated Measures ANOVA

### 개념
동일한 실험 대상으로 부터 시간, 처리등 조건을 달리하여 반복 측정한 자료
### 주요 분석 목적
- 집단 간 (대조군 vs 실험군) 평균 반응 프로파일 차이 검정
- 시간에 다른 변화패턴을 파악하고 집단간 평균 반응의 차이를 검정한다.
### 특징
- 한 개체에서 얻은 관측치들이 서로 상관됨
- 결측치 발생 가능성이 높음 ( 무응답, 탈락 등 )
- 개인차에 대한 노이즈가 줄어들고 시간, 처리 효과를 명확하게 파악할 수 있다.

$\therefore$  단순 ANOVA 로는 검정이 어렵다
### 예시
- 시간 : 동일 환자의 혈압을 0주, 4주, 8주에 반복측정
- 처리 : 동일 개체에 약의 농도를 증가시켜 가면서 반응값 측정

##### 반복 요인이 하나인 반복측정 자료의 형태

| 처리       | 개체        | t1                      | t2                      | $\cdots$ | tK                      |
| -------- | --------- | ----------------------- | ----------------------- | -------- | ----------------------- |
| 1        | 1 ~ $n_1$ | $y_{111}$ ~ $y_{1n_11}$ | $y_{I12}$ ~ $y_{In_I2}$ | $\cdots$ | $y_{11K}$ ~ $y_{In_1K}$ |
| 2        | 1 ~ $n_2$ | $y_{211}$ ~ $y_{2n_21}$ | $y_{212}$ ~ $y_{2n_22}$ | $\cdots$ | $y_{21K}$ ~ $y_{2n_2K}$ |
| $\vdots$ | $\vdots$  | $\vdots$                | $\vdots$                | $\vdots$ | $\vdots$                |
| I        | 1 ~ $n_I$ | $y_{I11}$ ~ $y_{In_I1}$ | $y_{I12}$ ~ $y_{In_I2}$ | $\cdots$ | $y_{I1K}$ ~ $y_{In_1K}$ |
##### 구형성 (Sphericity)
반복 요인 (수준 K) 에 대해 모든 시점 쌍의 차이 $Y_K - Y_l$ 의 분산이 동일
- "수준 간 차이" 를 보는데 필요한 정보(대비) 의 분산이 모두 동일
- 복합 대칭성 보다 완화된 조건
- F-stat 이 유효하기 위한 조건

구형성 가정 검정 : Mauchy's Test Of Sphericity
- 시간 간 차이들의 분산이 모두 같은가
	- 시점 간 차이를 만들어서 그 차이들의 분산이 모두 같은지 검정
- SAS 에서 printe 옵션으로 검정 가능

$H_0$ : 구형성이 성립한다.
$H_1$ : 구형성이 성립하지 않는다.

# 반복측정 자료분석을 위한 통계 모형

| 접근법         | 가정               | 특징                     |
| ----------- | ---------------- | ---------------------- |
| 단변량 분산분석    | 구형성 (Sphericity) | 단순하고 직관적, 결측치 처리 어려움   |
| 다변량 분산분석    | 다변량 정규성          | 구형성 위반시 대안, 검정력 낮음     |
| Mixed Model | 공분산 구조 명시        | 유연성 높고, 분균형과 결측치 처리 가능 |

## 반복측정 분산 분석 Repeated Measures ANOVA
보통의 분산분석 모형에 대체 효과를 나타내는 항을 추가한 모형
반복 요인이 하나인 경우 : 시간이나 조건이 하나의 축으로만 반복되는 상황
구형성 : 모든 시점간의 차이의 분산이 동일하다.

가장 전통적인 방법, 단변량 정규분포를 따른다.
구형성 가정시에서 진행된다.


$$
\begin{align}
y_{ijk}\;=\;\mu+\alpha_i+\beta_{ij}+\mathbb \gamma_k+\delta_{ik}+\epsilon_{ij}
\end{align}
$$

- $y_{ijk}$ : 처리 i 에서 j 번째 개체의 시점 k 에 측정한 반응 측정치
- $\mu$ : 반응 측정치 $y_{ijk}$ 의 전체 평균
- $\alpha_{i}$ : i 번째 처리의 효과
- $\beta_{ij}$ : 처리 i 에서 j 번재 개체의 효과
- $\gamma_k$ : 시첨 k 의 효과
- $\delta_{ik}$ : 처리 i 와 시점 k 의 교호작용 효과
- $\epsilon_{ij} = (\epsilon_{ij1} \cdots \epsilon_{ijK})$ : 오차항

### 검정 가능 종류
- 집단간 평균차이 검정
- 시간에 따른 변화 검정
- 시점과 처리간 상호작용 

### 모형 가정 
오차항 $\epsilon_{ij}$ 는 서로 독립이고 평균 0 인 다변량 정규분포
오차항의 공분산 행렬
- 구형성 (Sphericity) :
	각 시점간의 차이 ($\gamma_k$)가 모두 분산이 동일하다.
	이 가정이 만족되면 반복측정 ANOVA 를 쓸때 F-stat 이 올바르게 작동한다.
	- 복합 대칭성 (Compounded Symmetry) :
		모든 시점의 분산이 동일하고 시점간의 공분산 역시 일정하다는 가정
		시간의 간격이 달라도 상관의 크기가 변하지않는다.
		모든 시점이 서로 똑같이 상관되어있다.

#### 복합 대칭성 (Compounded Symmetry)
반복 측정 자료에서 개체들은 일정 시점마다 측정되기 때문에 각 반복 측정치들 간 상관관계가
존재하는데, 이 상관관계들이 모두 동일하다는 가정

- 공분산 행렬 형태
$$
\begin{align}
\sigma^2
\begin{bmatrix}
1 & p & \cdots & p \\
p & 1 &  & \vdots \\
\vdots & & \ddots & p \\
p & \cdots & p & 1
\end{bmatrix}
\end{align}
$$

- 모든 시점의 분산이 같다. ( 등분산 )
- 모슨 시점 쌍의 공분산(상관) 이 동일하다.
- 즉, 시간 간격 자체가 상관을 바꾸지 않은 상황 (강한 개인차 + 약한 시간 효과)
- 실제 데이터에서는 만족하는 경우가 매우 드물다.

## 다변량 분산분석 (MONOVA)
오차항이 다변량 정규분포를 따른다.
각 시점을 별도의 종속변수로 취급하고 전체를 다변량 정규분포로 모델링한다.
y 가 하나인 모델링만 계속했었지만, 다변량 분산분석은 y 가 k 개 존재한다.
종속변수를 한꺼번에 취급한다.

구형성 가정이 필요없고 ANOVA 의 대안 (MANOVA)
표본수가 적을 때 검정력이 낮다.
데이터가 충분히 많을때만 실질적인 장점을 가진다.

# 접근 방법

| 구분        | 단변량 접근                                                         |
| --------- | -------------------------------------------------------------- |
| 모형의<br>형태 | 각 시점의 반복값을 하나의 반응 변수로 취급<br>(즉, 하나의 모델에 여러측정치를 넣음)<br>$Y_{ij}$ |
| 오차항<br>가정 | 구형성 가정 필요 <br>(시간 간 차이의 분산 동일,공분산 구조 대칭)                       |
| 장점        | 계산 간단 직관적인 F 값                                                 |
| 단점        | 구형성 위배 시 결과 왜곡<br>GreenHouse-Geisser,Huynh-Feldt 보정 필요         |
| 검정<br>통계량 | F-검정 (단일 F)                                                    |

| 구분        | 다변량 접근                                                                           |
| --------- | -------------------------------------------------------------------------------- |
| 모형의<br>형태 | 각 시점을 별도의 종속변수 취급<br>(다변량 분산분석 수행)<br>$Y_i\;=\;(Y_{i1},Y_{i2},Y_{i3}\cdots)$     |
| 오차항<br>가정 | 공분산 가정 불필요<br>(공분산 구조 자유 허용)                                                     |
| 장점        | 구형성 위배시 정확, 일반 공분산 구조 허용                                                         |
| 단점        | 표본수가 작으면 검정력 감소 가능                                                               |
| 검정<br>통계량 | Wilk's Lambda<br>Pillai's Trace<br>Hotelling-Lawley Trace<br>Roy's Greatest Root |
# 모형 가정이 위반된 경우
- 구형성이 깨진다.
	- 즉, 시점간의 상관이 일정하지 않다.
	- 어떤 시점 쌍은 상관이 높고(변동 작음), 어떤 시점 쌍은 상관이 낮다 (변동 큼)
	- 시점간 대비마다 신뢰성이 다르다.
	- 시간 효과의 평균 제곱합이 일정하지 않은 오차항을 섞어서 계산하게 된다.
	- F 분포가 더이상 정확히 F 분포를 따르지 않음, 보통 작은 p-value 를 계산함
		- Type 1 Error 증가
	- 실제 자료에서 복합대칭성 또는 구형성을 만족하기 힘들다.

- 따라서, 구형성 가정이 만족하지 않는다면 다변량 접근방법 (MONOVA) 사용
- GH, HF 를 통한 자유도를 보정한 후 p-value 을 수정하는 방법도 존재한다.

# 구형성을 만족하는 경우 반복측정 분산분석표
## 가설
###### 구형성을 만족하는 자료의 분산분석표

| 요인     | 자유도          | 제곱합                                                                                                 | 평균제곱합                 | F     |
| ------ | ------------ | --------------------------------------------------------------------------------------------------- | --------------------- | ----- |
| 처리     | $I-1$        | $SSI=K\sum_{i=1}^I n_i(\bar y_{i..}-\bar y_{...})^2$                                                | $SSI/(I-1)$           | $F_1$ |
| 개체(처리) | $N-I$        | $SSJ=K\sum_{i=1}^I \sum_{j=1}^{n_i}(\bar y_{ij.}-\bar y_{i..})^2$                                   | $SSJ/(N-I)$           |       |
| 시간     | $K-1$        | $SSK=N\sum_{k=1}^K n_i(\bar y_{..k}-\bar y_{...})^2$                                                | $SSK/(K-1)$           | $F_2$ |
| 처리x식나  | $(I-1)(K-1)$ | $SS_{IxK}=\sum_{k=1}^K \sum_{i=1}^I n_i(\bar y_{i.k}-\bar y_{i..}-\bar y_{..k}+\bar y_{...})^2$     | $SS_{I*K}/(I-1)(K-1)$ | $F_3$ |
| 오차     | $(N-1)(K-1)$ | $SSE=\sum_{k=1}^K \sum_{i=1}^I \sum_{j=1}^{n_i} (y_{ijk}-\bar y_{ij.}-\bar y_{i.k}+\bar y_{i..})^2$ | $SSE/(N-I)(K-1)$      |       |
| 계      | $NK-1$       | $SST=\sum_{k=1}^K \sum_{i=1}^I \sum_{j=1}^{n_i} (y_{ijk}-\bar y_{...})^2$                           |                       |       |

$$
\begin{align}
F_1={SSI / (I-1) \over SSJ/(N-I)} \sim F(I-1,N-I)
\end{align}
$$
$$
\begin{align}
F_2={SSK / (K-1) \over SSE/(N-I)(K-1)} \sim F(K-1,(N-I)(K-1))
\end{align}
$$
$$
\begin{align}
F_3={SS_{IK} / (I-1)(K-1) \over SSE/(N-I)(K-1)} \sim F(I-1,(K-1)(N-I)(K-1))
\end{align}
$$


F 값은 오차항의 평균 제곱합으로 나누었다.
처리에 대한 효과를 검정할 때에, 개체내의 변동을 반영하여
F 값으로 집단간의 차이를 검정하게 된다.

### Between-subject effect (Treatment Effect)
$H_0$ : $\alpha_1=\alpha_2=\cdots=\alpha_I=0$

각 처리간 평균 반응 수준이 동일한가?
### Within-subject effects (Time Effect)
$H_0$ : $\gamma_1 = \gamma_2 = \cdots = \gamma_K = 0$

시간이 지나도 평균 반응이 일정한가?
### Interaction Between Treatment and Time
$H_0$ : $\delta_{11} = \delta_{12} = \cdots = \delta_{1K} = 0$

처리의 종류에 따라 시간 경과에 따른 반응 패턴이 다른가?


## 예제
고혈압 환자 66 명을 대상으로 32 명에게는 기존의 혈압강하제 (C) 를 투여하고
나머지 34명에게는 새로운 혈압강하게 (E) 를 투여하여, 각각 3회에 걸쳐 혈압을 측정하였다.
즉, 첫 방문시 약을 투여받기 전의 혈압과 4주후, 8주후의 혈압을 측정하였다.

| 처리 C 개체 | 0주후 | 4주후 | 8주후 | 처리 E 개체 | 0주후 | 4주후 | 8주후 |
| ------- | --- | --- | --- | ------- | --- | --- | --- |
| 1       |     |     |     | 1       |     |     |     |
| 2       |     |     |     | 2       |     |     |     |


1. 기존 약과 새로운 혈압 강하제 간 유의한 차이가 있는가?
2. 방문 시간에 따라 혈압에 차이가 있는가?
3. 약과 방문 시간 간에 교호작용이 있는가?


### 코드
각 개체간 시간은 서로 상관이 있으므로 반복측정 분산분석을 시행한다.
시간에 지남에 따라 혈압패턴이 두 약 사이에 다르게 나타나는가?

```SAS
proc glm data=blood
	class trt
	model week0 week4 week8 = trt / nouni ss3;
	repeated week 3 (0 4 8) contrast(3) / sumamry printe;
run;
```

시간에 따른 변화(반복요인) 와 처리군 간 차이(집단요인) 를 함께 고려한 반복측정 ANOVA
- trt(처리군) 에 따라 시간 (week0 , week4, week8) 별 반응이 달라지는 지를 분석
- nouni : 각 시점 별로 따로 ANOVA 표를 출력하지 않도록 한다.
- ss3 : 제 3 종 제곱합 을 사용한다.
- contrast(3) : week8 을 기준으로 한다.
- profile : 인접한 시점간의 차이를 검정한다 ( week0 <-> week4, week4 <-> week8)
- polynomial : 시간 변화의 추세 (선형, 2차, 3차 곡선 등) 를 검정

시간이 지나면서 측정값들 사이의 상관이 점점 커진다.
- 특히 4주와 8주간의 관계가 가장 강함 : 시간에 따른 안정화된 효과나 치료의 지속적 영향

Mauchly's Test (마클리 검정) :
	$H_0$ : 구형성 가정이 성립한다.
	$H_1$ : 구형성 가정이 성립하지 않는다.

두 행 모두 p < 0.05 : 구형성 가정이 위배되었다.
- F 검정 시 자유도 보정 (Greenhouse-Geisser, Huynh-Feldt 등) 적용
