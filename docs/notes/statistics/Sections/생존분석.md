# Survival Analysis
# 개요
- 생존분석 (Survival Analysis)
사건이 발생가기 까지의 시간을 분석하는 통계기법

- 주요 관심 
	- 특정 시점까지 생존할 확률
		- 1년 생존률, 5년 생존률, 환자 절반이 생존하는 시간(중앙생존시간)
	- 두 그룹간 생존시간 분포 차이
	- 생존에 영향을 미치는 요인 (나이, 성별, 기저질환)

- 응용분야 : 의학 (사망,처리에 대한 반응, 질병의 재발), 공학 (부품수명) 등
## 생존시간 / 중도절단
- 생존시간 (T) : 사건 발생까지의 시간
- 중도절단 (Censoring) : 사건이 연구에서 이탈한 경우
	- 제 1형 (고정 종료) : 연구 종료시점이 정해져 있음
	- 제 2형 (고정 실패 수) : 사전에 관측할 사망자 수를 미리 정함
	- 제 3형 (임의 절단) : 연구자가 통제 불가능한 이탈
## 모형
$$
\begin{align}
&X_i = min(T_i,C_i) \\\\
\delta_i
&\begin{cases}
&1\quad T_i \text{기 중도절단되지 않은 경우} \\
&2\quad T_i \text{중도 절단된 경우}
\end{cases}
\end{align}
$$
- $T_i$ , $C_i$ 는 서로 독립
## 함수
$$
\begin{align}
&S(t)=P(T>t)=1-F(t) \\
&h(t)=\lim_{dt\rightarrow0}P(t<T\leq t+dt\;|\;T\geq t)/dt = f(t)/S(t)
\end{align}
$$

- 생존함수 $S(t)$ : t 시간 이상 생존할 확률
	- 추정 :
		- 비모수정 : 분포 가정 없이 경험적으로 추정
		- 모수정 : 특정 분포 (지수, Weibull) 가정
		- 준 모수적방법 : Cox 비례위험모형

- 생존시간 T 의 위험 함수 (Hazard) $h(t)$ :
	- t 시점까지는 생존했다가 t 시점 바로 직후 사망하게 되는 순간 위험률

## 비모수적 방법 개요
- 중도절단이 없는 경우

생존시간 $x_1,x_2,\cdots,x_n$ 을 순서대로 배열 : $t_1 < t_2 < \cdots < t_n$
경험적 분포 함수 (Empirical Distribution Function)
$S(t_i)=P(T>t_i)$ : 
- 생존시간이 $t_i$ 이상인 관측치 수 / 총 관측치 수
- 1 - ($t_i$ 까지의 사망자 수) / n

- 중도절단이 있는 경우
	- 생명표 방법 (Life-table) : 환자수가 상당히 많을 때 관측자료를 시간의 단위로
		묶어 추정
	- 누적 한계 추정법 (Product-Limit Method) : 환자 개개인의 관측 자료의 시점에서 추정, 생존곡선을 시각화 할 수 있다.

### 생명표 방법
- 개인별 측정이 불가능하다.
- 중도 절단이 있는 경우
- 구간별 요약 형태
- 유효 인원 수를 계산하는 것이 중요하다.
- 생존 시간을 k 개의 구간으로 나누어 계산
$$
\begin{align}
(t_{i-1},t_i]\quad(i=1,\cdots,k)\quad(t_0=0)\quad(t_{k+1}=\infty)
\end{align}
$$

| 구간               | 사망자수     | 중도절단수    | 위험 그룹 인원수 | 유효인원수    | 사망률        | 생존율        | 생존함수          |
| ---------------- | -------- | -------- | --------- | -------- | ---------- | ---------- | ------------- |
| $(t_{0}, t_1]$   | $d_1$    | $c_1$    | $n_1$     | $n'_1$   | $\hat q_1$ | $\hat p_1$ | $\hat S(t_1)$ |
| $(t_{1}, t_2]$   | $d_2$    | $c_2$    | $n_2$     | $n'_2$   | $\hat q_2$ | $\hat p_2$ | $\hat S(t_2)$ |
| $\vdots$         | $\vdots$ | $\vdots$ | $\vdots$  | $\vdots$ | $\vdots$   | $\vdots$   | $\vdots$      |
| $(t_{i-1}, t_i]$ | $d_i$    | $c_i$    | $n_i$     | $n'_i$   | $\hat q_i$ | $\hat p_j$ | $\hat S(t_i)$ |
| $\vdots$         | $\vdots$ | $\vdots$ | $\vdots$  | $\vdots$ | $\vdots$   | $\vdots$   | $\vdots$      |

- $n_i$ : i 번째 구간 시작 시점에서 생존사 수 $(n_i = n_{i-1} - d_{i-1} - c_{i-1})$
	- $n_i'=n_i - {c_i \over 2}$
- $d_i$ : i 번쩨 구간에서 사망자 수
- $c_i$ : i 번째 구간에서 중도 절단 인원 수
- $p_i$ : i 번째 구간 내에서의 생존율 ($q_i=1-p_i$)
	- $p_i=Pr(\text{i번째 구간에서 내내 생존}\;|\;\text{i번째 구간의 시작점에서 생존})=Pr(T>t_i\;|\;T>t_{i-1})$
- $S(t_i)$ = $P(T>t_i)$
	- = $P(T>t_i\;|\;T>t_{i-1})P(T>t_{i-1}\;|\;T>t_{i-2})\cdots P(T>t_1)$
	- $p_ip_{i-1}\;\cdots\;p_2p_1$
	- Kaplan-meier (KM) 방법과 유사하다.
- $\hat S(t_i)\;=\;\Pi_{k=1}^i(1- {d_k \over n_k'})$ 단, $\hat S(0)=1$ : 생존 함수 추정치
- $Var(\hat S(t_i))=\hat S^2(t_i)\sum_{k-1}^i {d_k \over n_k'(n_k' - d_k)}$ : 생존 함수 분산

## 비모수적 방법 : 누적한계 추정법 (Kaplan-Meier)
#### 생존함수 추정치
$$
\begin{align}
\hat S(t)=\Pi_{t_i < t}(1 - {d_i \over n_i})
\end{align}
$$
#### 생존 함수 분산 (Greenwood)
$$
\begin{align}
Var(\hat S(t_i))=\hat S^2(t_i)\sum_{t_i \leq t} {d_i \over n_i(n_i-d_i)}
\end{align}
$$

#### 특징 
- 일치성, 근사적 정규성 만족, 표본 클 때 통계적 성질 우수
- 중도절단 시간에는 변하지 않고 사망이 관측된 시점에서 변화
- 사망과 중도 절단이 동시에 발생한 경우 사망이 중도 절단보다 먼저 관측된 것으로 봄
- 마지막 생존 시간이 중도 절단되 경우 그 이후 시간에 대한 추정치 정의하지 않음

### 비모수적 방법을 이용한 생존 함수의 비교
중도 절단이 포함되어 있는 경우 두 그룹의 생존 분포 비교

첫번째 그룹
- 생존 시간 : $T_{11}$, $T_{12}$, $\cdots$ , $T_{1n_1}$ ~ $S_1$ $i.i.d$
- 중도 절단 시간 :  $C_{11}$, $C_{12}$, $\cdots$ , $C_{1n_1}$ ~ $G_1$ $i.i.d$

두번째 그룹
- 생존 시간 : $T_{21}$, $T_{22}$, $\cdots$ , $T_{2n_2}$ ~ $S_2$ $i.i.d$
- 중도 절단 시간 :  $C_{21}$, $C_{22}$, $\cdots$ , $C_{2n_2}$ ~ $G_2$ $i.i.d$

관측자료 ($X_{ij}=min(T_{ij}, C_{ij}))\;,\;\delta_{ij}=I(T_{ij}<C_{ij}$)
- 첫번째 그룹 : ($X_{11},\delta_{11}$) , ($X_{12},\delta_{12}$) , $\cdots$ , ($X_{1n_1},\delta_{1n_1}$)
- 두번째 그룹 : ($X_{21},\delta_{21}$) , ($X_{22},\delta_{22}$) , $\cdots$ , ($X_{2n_2},\delta_{2n_2}$)

## 비모수적 방법을 이용한 생존 함수의 비교 : 특정시점 $t_k$
한계점 :
- 특정시점 $t_k$ 만 비교하므로 생존 곡선의 전체를 비교하기에 적합하지 않다.
- 그룹간 차이가 어떤 시간대에 발생하는지 반영할 수 없다.
- 대안으로 Log-rank 검정을 사용하게된다.

가설
- $H_0$ : $S_1(t_k)$ = $S_2(t_K)$
- $H_1$ : $S_1(t_k)$ $\neq$ $S_2(t_K)$

검정통계량
$$
\begin{align}
Z={\hat S_1(t_k) - \hat S_2(t_k) \over \sqrt{\hat Var(\hat S_1(t_k)}+\hat Var(\hat S_2(t_k))}
\end{align}
$$

## 비모수적 방법을 이용한 생존함수의 비교 : 총괄적 비교 (Log-rank)
가설
- $H_0$ : $S_1$ = $S_2$
- $H_1$ : $S_1$ $\neq$ $S_2$

Mantel-Haenzel Log rank 검정법
$t_1 < t_2 < \cdots < t_k$ : 두 그룹 자료를 섞어 순서대로 나열한 사망 시간
모든 $t_i$ 에서 2차원 분할표 작성 ( 시점 : $i$ )

|      | 사망       | 생존              | 계        |
| ---- | -------- | --------------- | -------- |
| 그룹 1 | $d_{1i}$ | $n_{1i}-d_{1i}$ | $n_{1i}$ |
| 그룹 2 | $d_{2i}$ | $n_{2i}-d_{2i}$ | $n_{2i}$ |
| 계    | $d_{i}$  | $n_{i}-d_{i}$   | $n_{i}$  |

- $n_{1i}, n_{2i}, d_i$ 가 고정되었다고 하면 귀무가설 하에서 $d_{1i}$ 의 조건부 분포는 초기하분포

$$
\begin{align}
&E(d_{1i})=E_{1i}={n_{1i}d_{i} \over n_i} \\\\
&Var(d_{1i})=V_{1i}={n_{1i}n_{2i} \over n_{i}-1}{n_{i}d_{i} \over n_{i}^2} \\\\
&T={\sum_{i=1}^k (d_{1i} - E_{1i})^2 \over \sum_{i=1}^k Var_{1i}} \sim \chi_1^2 \\\\
&Z^2 \sim \chi_1^2\quad,\quad \sqrt{T}=Z\sim N(0,1)
\end{align}
$$

단측 검정을 하고자 할 때는 $\sqrt{T} \sim Z$ 를 사용한다.

## 비모수적 방법을 이용한 생존함수의 비교 : 총괄적 비교 (Gehan)
- 가중 Log-rank 검정
- 위험집단이 크면 클 수록 , 초기시점 일 수록 가중치가 더 크게 부여된다.
- 초기 시점에서 그룹 간 차이에 더 민감하게 반응하는 검정 방법
- 각 시점에서 위험에 처해있는 인원수에 비례하는 가중치 $w_i={n_i \over N_{+1}}$

$$
\begin{align}
T={\sum_{i=1}^k w_i(d_{1i} - E_{1i})^2 \over \sum_{i=1}^k w_i^2 Var_{1i}} \sim \chi_1^2
\end{align}
$$

- 두 그룹의 위험률의 비가 상수 일때 로그 순위 검정법 선호
- 두 생존 함수가 서로 교차하는 경우에는 로그 순위 검정법은 부적합

## 준모수적(Semi) 방법을 이용한 생존 함수의 추정과 비교 : Cox Model
- 생존시간에 대한 분포 가정 없이 로그위험함수를 공변량의 선형함수로 모형화 하여 추정
- Cox regression 으로 불림
- 위험함수들 간에 비례관계만 명시할 뿐 완전한 확률모형이 아님

생존시간 $T$ 에 영향을 주는 $k$ 개의 공변량 $x_1,x_2,\cdots,x_k$ 가 있을때, 위험함수
$h_0(t)$ 는 기저위험함수 : 공변량의 값이 모두 0 일때 위험함수의 값
$$
\begin{align}
\log h(t) = \log h_0(t)+\beta_1x_1 + \cdots + \beta_kx_k =\alpha(t) + \beta_1x_1 + \cdots + \beta_kx_k
\end{align}
$$
$\alpha(t)$ 의 예 (모수적 모델) - 시간에 따라서 증가하거나 일정할 수 있다.
- 지수분포 : $\alpha(t)$ = $\alpha_0$
- 곰페르츠분포 : $\alpha(t)$ = $\alpha_0$ + $\alpha t$
- 와이블분포 : $\alpha(t)$ = $h_0+h\log t$

- cox model 에서는 $\alpha(t)$ 를 모수적으로 지정하지 않고 남겨둔다는 것이 장점이다.

비례 위험 모형
$$
\begin{align}
{h_i(t) \over h_j(t)} = \exp[\beta_1 (x_{i1} - x_{j1}) + \cdots + \beta_k(x_{ik-x_{jk}})]
\end{align}
$$
- i번째 j번째 환자의 위험함수의 비가 $t$ 와 무관한 상수
- 함수 $h$ 는 $t$ 에 의존적이지만 결과는 $t$ 에 의존적이지 않다.

비례위험 가정 확인
- 두 환자에 대해 시간에 따른 $\log \hat h(t)$ 또는 $\log(\log \hat S(t))$ 그래프가 평행인지 확인

Remarks
- 완전한 우도함수를 유도할 수 없어서 회귀계수의 추정을 위해
	편우도(Partial Likelihood) 이용
- 편우도 함수를 이용한 추정은 순위에만 의존하는 비모수적방법이지만,
	완전한 우도함수에 비해 효율성이 크게 떨어지지 않음
