# Kraken

# 정의
Kraken 2는 메타유전체학(Metagenomics) 분야에서 환경이나 임상 샘플 등으로부터 얻은 짧은 DNA 서열(Read)이 어떤 미생물 유래인지 빠르고 정확하게 분류해 주는 생물정보학 소프트웨어입니다.

## 작동 원리
Kraken 2는 입력된 DNA 서열을 'k-mer'라는 매우 짧은 조각 ( ex. 150-mer )으로 쪼개어 분석합니다.

이렇게 추출된 k-mer들을 사전에 구축된 참조 유전체 데이터베이스와 비교하여 정확히 일치하는 서열을 찾습니다.

일치하는 k-mer가 발견되면, 해당 k-mer를 공통으로 가지고 있는 모든 유전체의 '가장 최근 공통 조상(Lowest Common Ancestor, LCA)'에 분류 값을 할당하는 방식으로 서열의 최종 분류군을 결정합니다.

## 연계 분석 도구
Kraken 2는 주어진 서열을 어떤 미생물 분류군에 속하는지 빠르게 분류하는 데 특화된 도구이지만, 유사한 서열이 많은 경우 종(species)이나 속(genus) 수준의 실제 상대적 풍부도를 정확하게 추정하는 데에는 한계가 있습니다.

따라서 샘플 내 미생물 군집의 종·속 수준 비율을 더 정밀하게 파악하려면, Kraken 2의 분류 결과를 바탕으로 베이지안 재추정을 수행하는 후처리 도구인 Bracken(Bayesian Reestimation of Abundance with KrakEN)을 함께 사용해야합니다.
