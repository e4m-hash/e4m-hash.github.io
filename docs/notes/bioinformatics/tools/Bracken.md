# Bracken

# 정의
Bracken(Bayesian Reestimation of Abundance with KrakEN)은
메타게놈(Metagenomics) 데이터에서 미생물 종(Species)이나 속(Genus) 수준의 상대적 풍부도(Abundance)를 고정밀로 재추정해 주는 생물정보학 통계 도구입니다.

주로 매우 빠른 서열 분류기인 Kraken2 의 분석 결과를 바탕으로, 더 정확한 미생물 군집 비율을 계산하는 데 사용됩니다.

## 개발 배경 및 목적
Kraken 알고리즘은 DNA 서열을 데이터베이스와 비교해 가장 잘 맞는 분류군에 매우 빠르게 할당하지만, 여러 종이 유사한 서열을 공유할 경우 종(Species) 단위가 아닌 상위 계통(예: 속, 과)으로 서열을 뭉뚱그려 분류하는 한계가 있습니다.

Bracken은 이러한 문제를 해결하기 위해 개발되었습니다.
Kraken이 1차적으로 분류한 결과를 가져와 상위 계통으로 할당된 서열들을
베이지안(Bayesian) 통계 기법을 사용해 다시 하위 종 단위로 정확하게 재분배하여 실제 군집 비율을 추정합니다.

## 핵심 작동 원리
Bracken은 데이터베이스 내의 유전체들이 서로 얼마나 동일한 서열을 공유하고 있는지에 대한 사전 확률 정보를 활용합니다.

서열이 종(Species)보다 높은 상위 노드에 할당된 경우 해당 서열들을 하위 종 노드들로 확률적으로 분배하며, 반대로 균주(Strain) 수준으로 할당된 서열들은 상위인 종 노드로 끌어올려 통합합니다.

이를 통해 샘플 내에 유전적으로 매우 유사한 여러 종이 섞여 있더라도 각 종의 정확한 서열 수와 비율을 계산해 낼 수 있습니다.

## 주요 특징
- 여러 유사 종이 존재하는 복잡한 메타게놈 샘플에서도 종 및 속 수준의 풍부도를 정확하게 추정합니다.
- 연구자의 목적에 따라 종, 속, 문 등 원하는 분류학적 수준을 지정하여 결과를 도출하는 유연성을 제공합니다.    
- 분류 속도가 압도적으로 빠른 Kraken 파이프라인에 이어서 실행되므로 전체 대용량 데이터의 분석을 신속하게 처리할 수 있습니다.
