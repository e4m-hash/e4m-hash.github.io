# MetaPhlAn

# 정의

MetaPhlAn4(Metagenomic Phylogenetic Analysis 4)는
메타게놈 샷건 시퀀싱(Shotgun Metagenomic Sequencing) 데이터를 분석하여
세균, 고세균, 진핵생물 등 미생물 군집의 분류학적 조성을 종(species) 수준에서
정밀하게 프로파일링하는 생물정보학 분석 도구입니다.
16S rRNA 데이터가 아닌 전체 유전체 데이터를 기반으로 하며,
특정 계통에만 존재하는 마커 유전자를 활용하여 
미생물의 상대적 풍부도(relative abundance)를 정확하게 추정합니다.
## 주요 특징
- 높은 해상도 및 분류 정확도: 세균, 고세균, 진핵생물에 대해
	SGB(Species-level Genome Bin) 수준의 정밀한 해상도를 제공하며,
	명확한 분류학적 할당을 수행합니다.

- 광범위한 마커 데이터베이스: 약 100만 개의 미생물 유전체(참조 유전체 및 메타게놈 조립 유전체)에서 추출한 약 510만 개의 고유한 계통 특이적 마커 유전자를 포함하고 있습니다.

- StrainPhlAn 연계 분석: StrainPhlAn 도구와 결합하여 미생물을 단순한 종 단위가 아닌 균주(strain) 수준에서 식별하고 추적할 수 있습니다.

- 뛰어난 효율성: 기존 분석 방식들에 비해 속도가 매우 빠르며, 민감도(sensitivity)보다는 특이성(specificity)을 우선시하여 잘못된 매핑(cross-mapping)으로 인한 오류를 최소화합니다.

## 작동 원리
1. 마커 유전자 선별: 각 계통(종, 속, 과 등)을 고유하게 식별할 수 있는 마커 유전자 데이터베이스(ChocoPhlAn)를 구축하여 사용합니다.

2. 리드 매핑(Alignment): 시퀀싱 리드(read)를 Bowtie2 알고리즘을 이용해 마커 유전자 데이터베이스에 매핑합니다.

3. 분류 및 풍부도 계산: 마커 유전자에 정상적으로 매핑된 리드만을 분석에 활용하여 군집 내 미생물의 상대적 비율(%)을 계산하며, 매핑되지 않은 리드는 '미분류(unclassified)'로 처리합니다.
