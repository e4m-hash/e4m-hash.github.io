# BBDuk

# 출처
[bbduk](https://bbmap.org/tools/bbduk)

# 정의
BBDuk (Decontamination Using Kmers)은 미국 에너지부(DOE) 산하 공동 유전체 연구소(JGI)에서 개발한 생물정보학 소프트웨어 패키지인 BBTools에 포함된 다기능 데이터 전처리 도구입니다.

차세대 염기서열 분석(NGS) 데이터의 품질을 높이기 위한 거의 모든 전처리 작업을 단일 프로세스 안에서 초고속으로 처리할 수 있도록 설계되었습니다.

## 주요 기능
BBDuk은 K-mer 매칭 알고리즘을 핵심으로 하여 다음과 같은 작업들을 한 번에 수행합니다.

- 어댑터 제거 (Adapter Trimming)

시퀀싱 과정에서 부착된 인공 어댑터 서열을 찾아 제거합니다.
참조 어댑터 서열 파일과 매칭되는 K-mer를 찾아내어 잘라내는 방식을 사용하며
양 말단(Paired-end) 리드가 겹치는 구간을 이용해 더욱 정확하게 어댑터를 식별할 수도 있습니다.

- 오염 서열 필터링 (Contaminant Filtering)

분석 목적과 맞지 않는 오염원(예: 미생물 메타게놈 분석 시 섞여 들어간 숙주의 인간 유전체 서열, PhX174 등)을 K-mer 매칭을 통해 데이터에서 완전히 걸러내거나 마스킹(Masking) 처리합니다.

- 품질 트리밍 및 필터링 (Quality Trimming & Filtering)

리드 말단의 품질 점수(Phred score)가 낮은 염기를 잘라내고(Trimming),
전체 리드의 평균 품질이 기준치 이하이거나 'N' 염기가 너무 많은 리드는 아예 버립니다(Filtering).

- 추가 기능

길이 기반 필터링(Length filtering), GC 함량 필터링, 데이터 포맷 변환(예: FASTQ ↔ FASTA), 복잡도(Entropy) 필터링, 다양한 품질 및 염기 분포 히스토그램 생성 등 NGS 전처리에 필요한 다채로운 기능들을 하나의 명령어로 통합 지원합니다.    

## 기술적 특징 및 장점

- 유연한 매칭 알고리즘: 단순히 완벽히 일치하는 서열만 찾는 것이 아니라, 해밍 거리(Hamming distance, 치환만 허용)나 편집 거리(Edit distance, 삽입/결실 포함) 파라미터를 조절하여 변이가 있거나 시퀀싱 오류가 포함된 어댑터/오염 서열도 유연하게 찾아낼 수 있습니다.
