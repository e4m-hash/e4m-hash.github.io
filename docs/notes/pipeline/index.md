# 파이프라인 · 재현성
수집한 데이터를 재현 가능한 워크플로로 처리하는 단계입니다.
같은 입력이면 누구나 같은 결과를 얻도록 자동화·컨테이너화합니다.

## 주제

- 워크플로 엔진 — Nextflow, nf-core
- 컨테이너 — Docker / Podman, Singularity
- 인프라 — Kubernetes, Linux 환경, 클라우드(AWS/GCP/Azure)
