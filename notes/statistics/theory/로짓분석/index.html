    Logit Analysis¶

    ID 반응변수 설명변수
    1 \(y_1\) \(X_{11},X_{21},\cdots,X_{k1}\)
    2 \(y_2\) \(X_{12},X_{22},\cdots,X_{k2}\)
    \(\cdots\) \(\cdots\) \(\cdots\)
    n \(y_n\) \(X_{1n},X_{2n},\cdots,X_{kn}\)
    • 범주형 종속변수 (특히 이항) 을 설명하기 위한 방법
    • 선형 회귀를 그대로 적용할 수 없으므로, p 를 logit 변환하여 모델링
    \[ \begin{align} logit(p)\;=\;\log(\frac{p}{1-p}) \\ -\infty \leq \log(\frac{p}{1-p}) \leq \infty \end{align} \]
    \[ \begin{align} p\rightarrow 0\;:\log(\frac{p}{1-p})\;=\;-\infty \\ p\rightarrow 1\;:\log(\frac{p}{1-p})\;=\;\infty \\ \end{align} \]

    로지스틱 회귀분석 (Logistic Regression)¶

    • 독립변수 X 가 종속변수 Y \(\in\) {0,1} (이항) 에 미치는 영향을 추정
    \[ \begin{align} logit(p)\;=\;\log(\frac{p}{1-p})\;=\;\beta_0+\beta_1X_1+\cdots+\beta_kX_k \end{align} \]

    반응변수가 이항인 경우¶

    • 설명변수 :
      • 체중, 나이 등 수치자료
      • 처리의 종류 (대조/실험군)
      • 종양의 크기 (대/중/소)
    • 반응변수 :
      • 2개의 가능한 값 (예/아니오, 생존/사망 등)

    이항반응 변수의 경우¶

    종속변수는 베르누이 분포를 따른다.

    \[ \begin{align} Y\sim\text{Bernoulli}(p) \end{align} \]

    확률분포 식 : y 는 이항분포를 따른다. - \(P(Y=y)\;=\;p^y(1-p)^{1-y}\;,\;y={0,1}\) Y 함수의 기댓값 = 성공으로 정의된 확률변수 p - \(E(Y)\;=\;0\;*\;Pr(Y=0)\;+\;1\;*\;Pr(Y=1)\;=\;Pr(Y=1)\;=\;p\)

    반응 변수 Y 의 성공확률(p)에 영향을 미치는 설명변수 X 들을 고려 - \(Y|X\sim \text{Bernoulli}(p_x)\) 확률밀도 함수 - \(P(Y=y|X=x)\;=\;p_x^y(1-p_x)^{1-y},\;y={0,1}\) - 조건부 분포를 쓰는 이유는 특정 설명 변수 X 가 주어졌을때 성공확률이 얼마인지 계산하기 위해

    성공확률에 영향을 미치는 설명변수를 고려하는 법¶

    선형 모형¶

    \[ \begin{align} p_x\;=\;\alpha+\beta x \end{align} \]
    • 성공확률 \(p_x\) 가 0 과 1 을 벗어날 수 있다. (확률 정의 위반)
    • 선형회귀 에서 x 값이 변해도 분산은 비슷해야한다. (선형회귀 기본 가정-등분산성 위반) 베르누이 변수의 분산(\(p_x(1-p_x)\))은 x 에 따라 달라진다. 극단 영역에서 둔감, 중간에서 가장 민감하다. 선형 모형에서 발생하는 다양한 문제들을 해결하기 위해 logit 모형(비선형) 을 사용

    비선형 모형 (expit 함수) -> (logit 함수)¶

    \[ \begin{align} &p_x\;=\;{\exp(\alpha+\beta x) \over 1+\exp(\alpha+\beta x)} \\\\ &p_x\;+\;p_x \exp(\alpha+\beta x)\;=\;\exp(\alpha+\beta x) \\\\ &{p_x \over \exp(\alpha+\beta x)} + p_x\;=\;1 \\\\ &\exp(\alpha+\beta x)(1-p_x)\;=\;p_x \\\\ &\exp(\alpha+\beta x)\;=\;{p_x \over 1-p_x} \\\\ \text{logit}\rightarrow\quad&\alpha + \beta x\;=\;\log{p_x \over 1-p_x}\text{(log odds)} \end{align} \]
    \[ \begin{align} \log{p_x \over 1-p_x}\;=\;\alpha+\beta_x\;,\; \quad p_x\in \{0,1\} \end{align} \]

    • 기울기 : \(p_x\) 가 0.5에서 가장 크고 끝으로 갈수록 작아진다. (S자 반응)

    • \(\beta > 0\) : 증가하는 곡선

    • \(\beta = 0\) : 성공확률이 설명변수와 무관
    • \(\beta < 0\) : 감소하는 곡선
    • 곡선의 변화율은 \(\vert \beta \vert\) 에 비례
      • 완만할수록 (유선) \(\beta\) 가 작고
      • 급격할수록 (계단) \(\beta\) 가 크다.

    x 가 한단위 증가할때는 odds(성공확률/실패확률) 가 \(\exp(\beta)\) 만큼 변한다. \(Y \vert X \approx \text{bernulli}(p_x)\)

    • 성공할 오즈(odds)
    \[ \begin{align} {p_x \over 1-p_x}\;=\;e^{\alpha+\beta x} \end{align} \]
    • 설명변수 x 가 한단위 증가할때마다, 오즈는 \(e^\beta\) 배 증가
    • 로그 오즈 ( 로짓 모형 )
    \[ \begin{align} \log{p_x \over 1-p_x}\;=\;\alpha+\beta_x\;,\; \quad p_x\in \{0,1\} \end{align} \]
    • k 개의 설명변수 \(X_1,\cdots,X_k\) 가 있는 로짓 모형
    \[ \begin{align} p_x\;=\;{e^{\alpha+\beta_1x_1+\beta_2x_2+\cdots+\beta_kx_k} \over 1 + e^{\alpha+\beta_1 x_1+\beta_2x_2+\cdots+\beta_k x_k}} \end{align} \]
    \[ \begin{align} \log{p_x \over 1-p_x}\;=\;\alpha+\beta_1x_1+\beta_2x_2+\cdots+\beta_kx_k \end{align} \]

    if) \(\beta=0.7\) 이면 odds 약 2배 증가

    최대 가능도 추정량 MLE¶

    \(\beta\) 추정

    • X=x 가 주어져 있을때, Y 의 조건부 확률 밀도 함수
    • \(P(Y=y | X=x)=p_x^y(1-p_x)^{1-y},y=0,1\)
    • 로짓 모형
    \[ \begin{align} p_x\;=\; {\exp(\alpha+\beta x) \over 1+\exp(\alpha+\beta x)} \rightleftarrows \log{p_x \over 1-p_x}\;=\;\alpha+\beta_x\;,\; \quad p_x\in \{0,1\} \end{align} \]
    • 가능도 함수
    \[ \begin{align} L(\alpha,\beta)\;=\;\Pi_{i=1}^nP(Y_i=y_i\;\vert\;X_i=x_i)\;=\;\Pi_{i=1}^np_{x_i}^{y_i}(1-p_{x_i})^{1-y_i} \\\\ =\Pi_{i=1}^n({\exp(\alpha+\beta x_i) \over1+\exp(\alpha+\beta x_i)})^{y_i}(1-{\exp(\alpha+\beta x_i) \over 1+ \exp(\alpha + \beta x_i)})^{1-y_i} \end{align} \]
    ID 반응변수 설명변수
    1 \(y_1\) \(X_{11},X_{21},\cdots,X_{k1}\)
    2 \(y_2\) \(X_{12},X_{22},\cdots,X_{k2}\)
    \(\cdots\) \(\cdots\) \(\cdots\)
    n \(y_n\) \(X_{1n},X_{2n},\cdots,X_{kn}\)

    각 데이터 행의 발생확률을 모두 곱해주는 것이다. 데이터가 실제로 관측될 확률 최대로 만드는 \(\alpha, \beta\) 로 표현한것이다. -> 뉴튼랩스 등 최적화 방법 을 통해 두 값을 찾는다.

    \(\therefore\) 현재 가진 데이터를 가장 잘 설명하는 계수를 찾는 방식

    로짓 모형의 검정¶

    • 설명 변수와 연관선
    • 회귀 계수의 유의성 검정
      • 가설 : \(H_0 : \beta_i = 0\) , \(H_1 : \beta_i \neq 0\)
        • 기울기가 0 인지 아닌지
        • 효과가 있다 vs 없다.
    • 오차항에 대한 가정을 쓰지 않는다. ( 오차항이 이항분포의 성질을 따른다. )
    • 검정 통계량이 t 분포에서 달라지게 된다.
    • Wald 검정 통계량
    \[ ({\hat \beta_i \over SE_{\hat \beta_i}}) \sim \chi^2(1) \]
    - if) $\hat p_i = -0.5 , SE = 0.2$
- 검정 통계량 : 0.25

    다항반응 변수의 경우¶

    • 순서형 반응 변수의 경우
      • 비례 오즈 (Proportional Odds) 로짓 모형 조건 : 모든 범주구간에 대해 같은 계수를 공유한다.

    반응변수 \(Y\) 가 \(J\) 개의 순서형 범주를 갖고 설명변수가 \(k\) 개 일때 \(i\) 번째 개체의 설명변수 벡터 \(x_i={x_{i1},x_{i2},\cdots,x_{ik}}\) 에 대하여 \(\pi_{ij}\) 를 \(i\) 번째 개체가 \(j\) 번째 반응을 나타낼 확률

    \(j\) 번째 누적확률은 반응변수 \(Y\) 가 범주 \(j\) 이하에 해당할 확률로 다음과 같이 정의한다.

    \[ \begin{align} F_{ij}=P(Y\leq j\;|\;x_i)=\pi_{i1}+\pi_{i2},\cdots,\pi_{ij}\quad(j=1,\cdots,J) \end{align} \]

    따라서 누적 로짓 은 다음과 같이 정의된다.

    \[ \begin{align} L_j=logit(F_{ij})=\log({F_{ij} \over 1-F_{ij} })=\log{\pi_{i1},\cdots,\pi_{ij} \over \pi_{ij+1},\cdots,\pi_{iJ}} \end{align} \]

    위 누적 로짓 모형에 설명변수 \(x_i\) 의 효과를 포함하여 아래와 같은 모형을 나타낸다.

    \[ \begin{align} L_j(x_i)=\alpha_j + \beta' x_i\quad(j=1,\cdots,J-1) \end{align} \]

    여기서 \(\beta\) 는 \(\beta=(\beta_1,\cdots,\beta_k)'\) 으로 표현되고 \(J-1\) 의 범주에 공통적으로 적용되는 벡터이다. 이것은 비례오즈 모형의 기본 가정이 된다. 이 식을 다시 쓰면

    \[ \begin{align} L_j(x_i)=\alpha_j + (\beta_1,\cdots,\beta_k)x_i=\alpha_j+\beta_1x_{i1}+\cdots+\beta_kx_{ik} \end{align} \]
    • 이웃 범주 (Adjacent-Categories) 로짓 모형

    • 연속비 (Continuation-Ratio) 로짓 모형

    • 명목형 반응 변수의 경우

      • 일반화 (Generalized) 로짓 모형 -> 기준범주를 정하여 나머지 범주와 비교한다.

    비례오즈에서 조건이 깨질 경우 일반화 로짓 모형을 선택한다.

    비례 오즈 로짓 모형¶

    • Y. J 개의 순서형 범주를 가지고 있는 반응 변수
    • (\(X_1\),\(\cdots\),\(X_K\)) : 설명변수 k개
      • \(x_i = (x_{i1},x_{i2},\cdots,x_{ij})^T\)
    • \(\pi_{ij}\;=\;Pr(Y=j|X=x_i)\) : i 번째 개체가 j 번)재 반응을 나타낼 확률
    • i 번째 개체가 j 번째 반응까지의 누적 확률
      • \(F_{ij} = P(Y \leq j | X = x_i) = \pi_{i1} + \pi_{i2} + \cdots + \pi_{ij},\quad j=\{1,\cdots,j\}\)
    모형¶
    • 누적 로짓 (Cumulative Logit)
    \[ \begin{align} L_j(x_i)\;=\;logit(F_{ij})\;&=\;\log{F_{ij} \over 1-F_{ij}},\;j=\{1,\cdots,j-1\} \\\\ &= \log {\pi_{i1}+\pi_{i2}+\cdots+\pi_{ij}} \\\\ &= \alpha_j +\beta_1x_{i1}+\beta_2x_{i2}+\cdots+\beta_kx_{ik} \end{align} \]
    • 비례 가정이 만족되지 않으면 일반화 로짓 모형으로 적합

    관절염 통증 - 남성과 여성을 2가지 처리 방법에 할당하여 통증의 개선 정도를 1) 전혀없음, 2) 약간 있음, 3) 상당히 있음 3가지의 순서형 반응으로 측정하였다. 통증의 개선도가 성별과 처리 방법에 따라 영향을 받는지 알아보고자 한다.

    명목형 반응변수 - 일반화 로짓 모형¶
    • 기준 범주 로짓
    \[ \begin{align} &\log({\pi_{ij} \over \pi_{ij}})=\alpha_j + \beta_{j1}x_{j1} + \beta_{j2}x_{j2}+\cdots+\beta_{jk}x_{jk}\quad(J=1,\cdots,J-1) \\\\ &=\beta_j^Tx_i \quad \text{단, } \beta_j^T=(\alpha_j,\beta_{j1},\cdots,\beta_{jk}) \end{align} \]
    \[ \begin{align} \pi_{ij}\;=\;{exp(\beta_j^T x_i) \over 1+\sum_{k=1}^{J-1}\exp(\beta_{k}^Tx_i)} \end{align} \]
    \[ \begin{align} \pi_{iJ}\;=\;{1 \over 1+\sum_{k=1}^{J-1}\exp(\beta_{k}^Tx_i)} \end{align} \]
