    Introduction¶

    생명과학 안에서의 통계학의 역할¶

    • 생명현상은 변이가 크고, 불확실성이 많음
    • 실험결과가 "우연인지, 실제 효과인지" 구분해야함
    • 연구 질문(가설 설정) -> 데이터 수집 -> 통계 분석 -> 해석 및 결론

    연구 목적¶

    모수 추정 Descriptive Purpose¶

    질병의 발생률, 유병률, 평균 생존 기간, 삶의 질 지표 기술/코호트 연구 활용

    인과 관계 규명¶

    특정 위험 요인이 질병 발생과 관련이 있는가? 관찰연구 (cohort , case-control) 활용

    중재 효과 평가¶

    새로운 치료법, 약물, 생활습관의 개입의 효과 및 안정성 검증 중재 -> 인위적 처리의 영향력 평가 무작위 배정 임상시험 (RCT) 활용

    임상 연구 유형¶

    종류 형식 시간성 방향 과거 현재 미래
    단면 관측 단면 단면 모든 정보 수집
    코호트 관측 종적 전향적 코호트구성,위험요인 평가 결과 관찰
    사례대조 관측 종적 후향적 위험요인평가 사례군 및 대조군 정의
    실험 실험 종적 전향적 중재 실시 결과 관찰

    단면연구¶

    • 특정 시점에서 질병 유병률, 위험요인 파악
    • 인과관계 규명에는 한계, 연관성 여부만 확인 가능
    • 일반적으로 빠르고 쉽게 수행 가능
    • 예: 설문조사, 센서스, 한국인의 HIV 발현률

    종적연구¶

    • 동일한 연구대상자들을 시간에 흐름에 따라 추적하는 연구
      • 전향적: 대부분 종적 연구
        • 특정 시점을 기준으로 연구대상자들을 시간의 흐름에 맞춰 앞으로 조사
      • 후향적
        • 연구대상자들을 모집 후 이들이 과거에 경험했던 요인들을 거꾸로 추적 조사

    관찰연구¶

    • 연구자가 개입 또는 중재 (Intervention)하지 않고,대상자의 특성과 결과를 관찰

    • 요인통제가 불가능. 따라서, 인과관계 추론에 한계 (혼란변수, bias 존재)

    • 사례-대조 연구 (Case-Control Study

      • 희귀 질병이나 경과기간이 긴 연구에 효율적으로 이용되는 후향적 연구
        • 예: 심장 발작을 겪은 사람들 (사례군) vs 그렇지 않은 사람들 (대조군)
        • 심장발작과 약물 남용간의 인과관계에 대한 추론

    • 코호트 연구 (Cohort Study) 같은 특성을 가진 집단을 일정기간 추적 조사하는 전향적 연구

      • 환자 선정 제어가 가능하여 관찰 연구 중 인과 관계 추론에 가장 이상적
      • 추적 중간에 사망이나 협조 거부 등으로 중도 탈락이 발생하여 연구관리에 어려움
      • 예: 비흡연자 집단 v.s. 흡연자 집단
      • 20년 추적 조사하여 폐암 발생 여부 조사로 흡연이 폐암에 영향을 미치는 영향 추론

    실험연구¶

    연구자가 의도적으로 개입하여, 관심있는 약물이나 또는 방법 등의 처리(treatment)를 연구 대상자에게 실시한 후 효과를 측정함으로써 처리와 반응간 인과관계 규명

    • 대조군과 비교하여 효과 검증 가능
    • 임상시험 (Clinical Trial)이 대표적 형태
    • 대조군 연구 (Controlled Trial): 대조군 (위약, 표준치료) 설정
    • 무작위배정 임상시험 (RCT): 무작위화, 눈가림을 통해 bias 최소화

    임상 시험 단계¶

    -> 최종 상용화 까지 평균 성공률 9.6%

    1상 ( Phase I )¶

    • 대상 : 소수의 건강한 지원자 ( 20 ~ 80 )
    • 목적 : 안정성 (Safety) , 독성 , 최대내약용량 (MTD) 확인
    • 성공률 : 60%

    2상 ( Phase II )¶

    • 대상 : 환자 수십 ~ 수백명
    • 목적 : 초기 유효성 (efficacy) , 적정용량 , 부작용 탐색
    • 소규모 무자귀 배정 시험 포함 가능
    • 성공률 : 30%

    3상 ( Phase III )¶

    • 대상 : 대규모 환자군 ( 수백 ~ 수천명 )
    • 목적 : 표준치료 대비 효과 및 안정성 비교 (허가 근거)
    • 무작위 배정 및 이중 눈가림 RCT
    • 성공률 : 60%

    4상 ( Phase IV )¶

    • 대상 : 시판 후 환자군
    • 목적 : 장기 효과 , 드물게 나타나는 부작용 파악
    • 약물감시, 실제 임상 적용 평가

    임상 시험 절차¶

    1. 계획 수립 & 승인요청
      • 임상시험계획서 작성
      • 식약처 , 기관윤리위원회 (IRB) 에 승인 신청
    2. 승인 심사
      • 계획서 검토
      • 승인 여부 결정
    3. 시험 의뢰 ( 책임 연구자 지정 )
      • 신약의 안정성, 효과 검증
    4. 피험자 동의 & 시험 진행
      • 과학적, 윤리적 기준에 따라 임상 시험 실시
    5. 실태 조사 ( 감독 )
      • 식약처가 임상시험 기관 및 연구자 대상으로 현장조사 실시
    6. 평가 & 허가
      • 식약처가 신약의 안정성과 유효성 평가 허가 후 판매 가능

    가설 검정 ( Hypothesis Testing )¶

    • 모수 ( Parameter )의 값에 대한 가설의 옳고 그름 판단 절차
    • 연구자의 연구 목적에 반하는 가정 하에 출발하여, 표본자료를 통해 연구자가 추구하는 것에 대한 증거를 찾고자 하는 일련의 방법론
    5% 유의수준에서 가설을 기각한다.
A 그룹과 B 그룹은 통계적으로 유의한 차이를 보였다 ( p < 0.01 )

    원칙¶

    • 대립가설 을 입증할 강력한 증거가 있을때 채택하고, 증거가 충분하지 못하면 연구자의 주장을 버리고 귀무가설로 돌아감

    • 귀무가설이 참이라는 가정하에서 표본이 관측된 값이 귀무가설에 대한 유의미한 반증이며 대립가설에 대한 입증이 확인된다면 귀무가설을 기각

    용어¶

    • 가설 : 귀무가설 (\(H_O\)) , 대립가설 ( \(H_A\) )
    • 오류 : 제 1종 오류 (\(\alpha\)) , 제 2종 오류 (\(\beta\))
    • 유의 수준
    • 유의 확률 (p-value)
    • 단측검정 ( one-sided test )
    • 양측검정 ( two-sided test )

    \(H_O\)¶

    • 기존의 사실, 주장
    • 연구자의 주장에 반하는 주장
    • 대립가설의 모수 영역의 반대

    \(H_A\)¶

    • 새로운 사실, 주장, 믿음
    • 연구자의 주장
    • 연구자가 입증하고자 하는 모수 영역

    유의확률 P-value¶

    • 귀무가설 하에서 관측된 사건 이상으로 귀무가설에 반하는 사건이 일어날 확률

    • p-value 가 작을수록 귀무가설에 반대되는 강력한 증거

    • 얼마나 작아야하는가?

      • 판단 기준 : 유의 수준
      • 예 : 유의 수준 5% -> p-value < 0.05 => 귀무가설 기각

    단측검정 (one-tail test)¶

    • 귀무가설에 대하여 작은값 또는 큰값 ( 즉, 한 방향으로 검정 )
      • \(H_O\) : 신약의 치료효과가 기존의 약과 차이가 없다.
      • \(H_A\) : 신약의 치료효과가 기존의 약보다 좋다.

    양측검정 (two-tail test)¶

    • 귀무가설에 대하여 양방향 (작은값, 큰값) 으로 검정하는 것
      • \(H_O\) : 신약과 기존의 약의 치료효과가 같다.
      • \(H_A\) : 신약과 기존의 약의 치료효과가 같지 않다.

    오류¶

    구분 실제
    \(H_O\) True \(H_O\) False
    \(H_O\) 기각 제 1종 오류 \(\alpha\)
    \(H_O\) 기각 실패 제 2종 오류 \(\beta\)

    • 제 1종 오류 : 귀무가설 \(H_O\) 가 사실인데 기각하는 결정을 내리는 오류

      • 예: 실제로 두 집단간의 치료 효과가 차이가 별로 없는데도 관측된 값에서 치료효과의 차이가 우연히 크게 나타날 수 있음

    • 제 2종 오류 : 귀무가설 \(H_O\) 가 거짓인데 기각하지 못하는 오류

      • 예: 치료효과의 차이가 있는데도 불구하고, 관측된 차이 우연히 매우 작은 값

    가설 검정의 원리¶

    • 유의수준 (Significant Level) : \(\alpha\) 가 일어날 확률의 최대 허용치
    • 검정력 ( Power ) : \(1-\beta\) -> 귀무가설이 사실이 아닐때 귀무가설을 기각하게 될 확률

    "제 1종 오류 확률을 고정시키고, 제 2종 오류확률을 최소화하는 검정방법(최강력 기각역) 을 찾아야한다."

    예제 1¶

    제약회사에서 두 약을 비교하기 위한 실험을 시행한 결과 신약의 치료율이 76%로 기존 약의 치료율 보다 6% 좋게 나타남
    - 치료 효과가 같은데도 우연히 더 좋게 나올 수 있다. - 신약의 치료 효과가 더 좋기 때문에 당연히 나타난 결과이다.

    -> 치료효과가 같을때 우연히 6% 이상의 차이를 보일 확률을 계산 -> 이 확률이 극히 작다면 귀무가설을 기각할 만한 충분한 증거가 된다.

    가설 검정의 절차¶

    1. 가설 설정
      1. 귀무가설 : 차이 효과가 없다
      2. 대립가설 : 차이 효과가 있다.
    2. 유의수준 설정
      1. 보통 0.05 사용
      2. 즉, 잘못된 결론 (\(\alpha\)) 를 허용할 최대 확률
    3. 검정통계량 계산
      1. 표본 자료를 이용해 통계량 산출
      2. ex.) Z , t, F 값 등
    4. 기각역 설정 & p-value 계산
      1. 기각역 (Critical Region) 설정
      2. 또는 p-value 를 계산하여 유의수준과 비교
    5. 결과 도출
      1. p-value 가 유의수준보다 작거나 같으면 귀무가설 기각
      2. p-value 가 유의수준보다 크면 귀무가설 기각 실패
    예 1¶
    새로운 다이어트 프로그램의 회원 모집 광고에서는 프로그램 참가자들이
5주 내에 평균 10kg 보다 많이 감량을 할 수 있다고 광고하고있다.
56명의 참가자들이 5주 동안의 감량 자료로부터 표본평균 & 표본표준편차를 계산한 결과
각각 10.7 Kg, 4.6 Kg 이었다고 하자.
이 자료에 따르면 광고의 문구가 사실이라고 할 수 있는가?
    1. 가설 설정
      1. 귀무가설 : 평균 감량 효과 10kg : \(\mu\;=\;10\)
      2. 대립가설 : 평균 감량 효과 10kg 초과 : \(\mu\;>\;10\)
    2. 유의수준 설정
      1. 보통 0.05 사용 (5%)
      2. 즉, 잘못된 결론 (\(\alpha\)) 를 허용할 최대 확률

    3. 검정통계량 계산

      \[ \begin{align} Z\;=\;\frac{X\;-\;\mu_o}{\sigma\sqrt{n}}\;\approx\;\frac{X-\mu_o}{\frac{s}{\sqrt n}}\;=\;\frac{10.7-10}{\frac{4.6}{\sqrt{5.6}}}\;=\;1.139 \end{align} \]

    4. 기각역 설정 & p-value 계산

      1. 기각역 (Critical Region) 설정 : \(Z \geq z_{0.05}\;=\;1.645\)
      2. p-value : \(P(Z\;\geq\;1.139)\;=\;0.1357\)
    5. 결과 도출
      1. p-value 가 유의수준보다 작거나 같으면 귀무가설 기각
      2. p-value 가 유의수준보다 크면 귀무가설 기각 실패 \(\therefore\) P 값이 유의수준보다 크므로 귀무가설 기각 실패 유의수준 5% 하에서 새로운 다이어트 프로그램은 5주내 평균 10kg 보다 많이 감량 할 수 있다고 볼 수 없다.

    • \(Z\;=\;1.139\)
    • \(Z\;\geq\;z_{0.05}\;=\;1.645\)
    • \(P(Z\;\geq\;1.139)\;=\;0.1357\)
    예 2¶

    다음 자료는 뚜렷한 폐기종 발전 이전의 흡연자에 대해 폐 파괴 현상을 조사한 것이다. 

    smoke = {16.6, 13.9, 11.3, 26.5, 17.4, 15.3, 15.8, 12.3, 18.6, 12.0, 24.1, 16.5, 21.8, 16.3, 23.4, 18.8}
non_smoke = {8.1, 6.0, 10.8, 11.0, 7.7, 17.9, 8.5, 13.0, 18.9}
    - 비흡엽자 집단과 호흡기질환 외의 다른 원인으로 사망한 흡연자 집단에 대해 폐 파괴 지수를 측정하였다. (높은 수치는 폐의 손상됨이 크다는 것을 뜻한다.) 각 그룹의 관측치들은 독립적으로 얻어진 것이며, 두 그룹에서의 모분산은 같다고 가정한다.

    생명과학 연구자료의 유형¶

    질적 자료 (Qualitative Data)¶

    • 범주형 (Catagorical) 반응 값이 몆개의 범주로 나누어진 것
      • 명목형 자료 (Nominal) - 범주의 순서가 없다. 예 : 남/여, 성공/실패, 흡연/비흡연, A/B/O/AB
      • 순서형 자료 (Ordinal) - 범주의 순서가 있다. 예 : 고통의 감소 정도 (전혀 없음 / 조금있음 / 매우 많음 등)

    양적 자료 (Quantitative Data)¶

    • 수치형 (Numerical)
      • 이산형 (Discrete) 예: 출산 횟수, 재발 횟수, 나이 등
      • 연속형 (Continuous) 예: 체중, 키, 혈압 등
